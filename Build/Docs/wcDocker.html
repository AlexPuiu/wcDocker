Class: wcDocker

wcDocker

The main docker instance. This manages all of the docking panels and user input. There should only be one instance of this, although it is not enforced.
See getting-started

new wcDocker(container, options)

Parameters:
Name Type Argument Description
container external:jQuery~selector | external:jQuery~Object | external:domNode A container element to store the contents of wcDocker.
options Object <optional>
 Options for constructing the instance.
Properties
Name Type Argument Default Description
themePath String <optional>
 'Themes' A folder path where all docker theme files can be found.
theme String <optional>
 'default' The active docker theme.
allowContextMenu Boolean <optional>
 true Overrides the default right click menu with ones that interact with docker.
hideOnResize Boolean <optional>
 false If true, panels will hide their contents as they are being resized.

function addDrawer(location) > wcDrawer

Adds a collapsible drawer container to a given position of the docking window. Drawer containers will appear on the outside of all currently docked panels. This should be used after you have finished laying out the main panels but before you dock any static panels at the top such as file or tool bar.

Parameters:
Name Type Description
