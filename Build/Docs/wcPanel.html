Class: wcPanel

wcPanel

The public interface for the docking panel, it contains a number of convenience functions and a layout that can be filled with a custom arrangement of elements.

new wcPanel(type, options)

PRIVATE - Use wcDocker.addPanel, wcDocker.removePanel, and wcDocker.movePanel to manage panels, this should never be constructed directly by the user.

Parameters:
Name Type Argument Description
type String The name identifier for the panel.
options wcPanel~options <optional>
 An options object passed from registration of the panel.
