Class: wcSplitter

wcSplitter

Splits an area in two, dividing it with a resize-able splitter bar. This is the same class used throughout docker to organize the docking interface, but it can also be used inside a panel as a custom widget. Note: The container needs to be positioned in either absolute or relative coordinates in css.
